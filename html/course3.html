Алгоритмы и структуры данных — это основа любого программирования. В этом курсе мы рассмотрим основные алгоритмы, такие как сортировка, поиск, и различные структуры данных, такие как массивы, списки и деревья.

Задание:

Напишите алгоритм для сортировки массива чисел по возрастанию (используйте алгоритм пузырьковой сортировки).

            
                // Пример кода
                let arr = [5, 3, 8, 4, 2];

                for (let i = 0; i < arr.length; i++) {
                    for (let j = 0; j < arr.length - 1; j++) {
                        if (arr[j] > arr[j + 1]) {
                            let temp = arr[j];
                            arr[j] = arr[j + 1];
                            arr[j + 1] = temp;
                        }
                    }
                }

                console.log(arr); // [2, 3, 4, 5, 8]
